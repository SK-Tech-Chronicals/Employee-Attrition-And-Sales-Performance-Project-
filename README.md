# 📊 Employee Attrition & Performance Dashboard (Excel)

---

## 📌 Project Overview

This project analyzes employee attrition and performance using Microsoft Excel.
The goal is to identify patterns behind employee exits and evaluate how factors like salary, department, and performance impact attrition.

## 🧠 Business Problem

* Why are employees leaving the company?
* Which department has the highest attrition?
* Does salary affect employee retention?
* How does performance relate to attrition?

---

## 📊 Dataset Details

The dataset includes:

* EmpID
* Department
* Region
* Salary
* Performance
* Experience
* Attrition (Yes/No)

---

## 🧹 Data Cleaning Steps

* Removed blank rows
* Fixed inconsistent values (Yes/No, text errors)
* Handled missing salary values
* Ensured proper data format

---

## ⚙️ Feature Engineering

Created additional columns:

* **Salary Band** → Low / Medium / High
* **Experience Level** → Fresher / Experienced
* **Attrition Flag** → Yes = 1, No = 0

---

## 📈 Key Performance Indicators (KPIs)

* **Total Employees**
* **Employees Left**
* **Attrition Rate (%)**
* **Average Salary**

---

## 📊 Charts Used

* Donut Chart → Attrition Distribution
* Column Chart → Department-wise Attrition
* Bar Chart → Salary Band Analysis
* Line Chart → Performance vs Attrition

---

## 🎛 Slicers (Interactive Filters)

* Department
* Region
* Salary Band
* Experience Level

---

## 🔍 Key Insights

* High attrition observed in specific departments
* Employees with lower salary tend to leave more
* Performance impacts retention trends
* Some regions show higher attrition
